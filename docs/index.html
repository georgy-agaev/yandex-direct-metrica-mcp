yandex-direct-metrica-mcp

MCP server for Yandex Direct + Yandex Metrica — reporting, joins, and a practical BI dashboard.

This project is not affiliated with, endorsed by, or sponsored by Yandex. Yandex, Yandex.Direct, Yandex.Metrica are trademarks of their respective owners.

What you get

  • Direct reporting (impressions/clicks/spend, campaign breakdowns)
  • Metrica exports (visits, goals/leads, sources)
  • Human-friendly joins: Direct vs Metrica by UTM / yclid (best-effort)
  • Dashboard generator (Option 1): HTML + JSON, including multi-account dashboards

Install (Claude Code)

claude mcp add yandex-direct-metrica-mcp -- \
  docker run --rm -i \
    --env-file /path/to/your/.env \
    -e MCP_ACCOUNTS_FILE=/data/accounts.json \
    -v /path/to/your/state:/data \
    ghcr.io/<owner>/yandex-direct-metrica-mcp:latest

Tip: for daily usage set date_to to yesterday (Direct/Metrica data for “today” is often incomplete).
