# Описание приложения для доступа к API Яндекс.Директа

## Технические данные
- Язык программирования: Python.
- Протокол: JSON (Direct API v5).
- Библиотеки: `tapi-yandex-direct 2021.5.29`, `tapi-yandex-metrika 2022.4.8`.

## Назначение приложения
Автоматизация отчетности и управления рекламой в Яндекс.Директ, с выгрузкой
поведенческой статистики из Яндекс.Метрики для аналитики и контроля эффективности.

## Основные функции
- Получение статистики и отчетов по кампаниям, группам, объявлениям, ключевым словам.
- Создание и обновление кампаний, групп, объявлений и ключевых слов.
- Работа со ставками и модификаторами (read-only на старте).
- Экспорт отчетов Метрики и при необходимости Logs API для склейки по UTM/yclid.

## Схема взаимодействия с Директом
1) Пользователь проходит OAuth авторизацию, приложение получает access/refresh token.
2) Приложение обращается к Direct API по HTTPS (JSON v5).
3) Данные обрабатываются локально, без хранения в сторонних сервисах.
4) При необходимости выполняется интеграция с Метрией для поведенческих метрик.

## Хранение и безопасность данных
- Токены хранятся локально в переменных окружения.
- Пользовательские данные не сохраняются в базе, только временная обработка.
- Для тестов используются Sandbox или тестовые аккаунты.

## Интерфейс
Публичного веб-интерфейса нет. Приложение запускается локально (CLI),
используется для автоматизации и выгрузок внутри компании.
